# 概要
二つのフォルダ間でファイルの同期処理を行うためのコマンドラインアプリ。
MSのフリーソフトSyncToysというアプリを使っていたが、不要なコピーをするなど、誤動作することが多く、その原因もわからない。
現在はこのアプリは非公開となってしまっており、オープンソースでもない。
また、同様な機能のオープンソースソフトもないようである。
このため、仕方なしに作成中。

危険なアプリでもあるので、開発は慎重に行っている。

実装も極力安全な処理を行うように気を付けている。

現時点では、まだ二つのフォルダの控えめな同期処理しか実現できていない。

片方のディレクトリでファイルが削除されていたら、もう片方のディレクトリからも削除するという機能はまだ実装できていない。
